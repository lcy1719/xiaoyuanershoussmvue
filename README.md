## 【项目采用技术】
架构: B/S、MVC
系统环境：Windows/Mac
开发环境：IDEA、JDK1.8、Maven、MySQL、Tomcat7.5/8/9
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、Vue、JavaScript、CSS、JavaWeb、HTML
## 【系统功能模块】
本系统包含管理员、商家、用户3个角色。
管理员角色：
1. 商家管理：管理商家基本信息，包括商家名称、联系方式、邮箱等。
2. 新闻信息管理：新增、修改和删除新闻内容。
3. 论坛管理：发布和管理论坛帖子，查看帖子回复。
4. 商品管理：管理商品基本信息，包括商品照片、库存、原价等。
5. 商品留言管理：查看和回复商品留言。
6. 商品订单管理：查看和管理商品订单信息。
7. 用户管理：管理用户信息。
商家角色：
1. 商品管理：管理商品基本信息，包括商品照片、库存、原价等，并可上下架商品。
2. 商品留言管理：查看和回复商品留言。
3. 商品订单管理：查看和管理商品订单信息。
4. 新闻信息管理：查看新闻内容。
用户角色：
1. 商品信息：查看商品信息，包括原价、现价、商家信息等，并可购买商品。
2. 确认下单：检查收货地址和商品信息，确认支付。
3. 商品订单：查看和管理自己的商品订单信息。
4. 购物车：将商品加入购物车并统一下单支付。
